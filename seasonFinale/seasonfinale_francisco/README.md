Gracias por usar el programa para visualizar el clima de Pancho (Francisco Barría).
-----------------------------------------------------------------------------------
La visualización incluye datos para ver temperatura, viento (vel) y humedad.
El programa ordena los datos desde izquierda a derecha como filas verticales,
donde los datos se ordenan de máximo a mínimo, desde abajo hacia arriba.

Para navegar por estos datos se necesita usar los botones predeterminados del programa.
---------------------------------------------------------------------------------------
En este programa, puedes visualizar el clima entre los valores que escojas.
Tan solo usa los botones de acceso rápido, o pon los valores de limite custom lo que necesites.


Visualización rápida:

  (Botón)		(Visualización)
	p 		      Año
	o		    Semestre
	i		      Mes
	u		    Semana
	y		      Día
--------------------------------------
	s               Cambio de data
	a		 Ocultar texto
--------------------------------------

Visualización custom:


	Limite Mínimo:
		Disminuir:  v
		Aumentar:   b

	Limite Mayor:
		Disminuir:  n
		Aumentar:   m


	Reset:    	    r
	Visualizar Custom   g
